# Next 13 Todo List 

Built with **Next.js** and **Prisma**.

## Features

* Used Prisma ORM with Mongo DB
* Complete CRUD
* Next Server Actions
* Next Auth with Github Provider
* Dark Mode

## How to use

```bash

# Step 1
git clone git@github.com:ChaitanyGhadigaonkar/Next-13-Todo.git

# Step 2
cd Next-13-Todo

# Step 3
npm install 

# or 

yarn 

# step 4
npm run dev 

or

yarn dev

```
